# Examples

## Single Specialist

```text
Spawn python_expert to refactor src/importer.py with type hints, pathlib, explicit exceptions, and pytest coverage.
```

## Review Workflow

```text
Spawn code_reviewer to review this branch against main for correctness, regressions, and missing tests. Report findings only.
```

## Security Audit

```text
Spawn security_auditor to audit the auth and billing routes for OWASP Top 10 risks. Prioritize exploitable issues and include remediation steps.
```

## Multi-Agent Planning

```text
Spawn orchestrator to plan a checkout feature. Have it identify which parts should go to api_designer, backend_developer, frontend_developer, database_architect, test_generator, and security_auditor.
```

## Buffer Workflow

```text
Spawn buffer_api to improve this launch announcement for LinkedIn and X, identify required Buffer channel IDs, and prepare a scheduled-post GraphQL mutation for next Monday at 10:00 UTC.
```
